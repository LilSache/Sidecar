# Security Policy

## Supported Versions

This project is currently in early alpha. Only the latest build is actively maintained.

| Version | Supported |
| ------- | --------- |
| v0.1.0-alpha (latest) | ✅ |
| Any older build | ❌ |

## Reporting a Vulnerability

This is a solo project maintained by one person. If you find a security issue please — email me directly as well:

📧 **prerollgames@gmail.com**

Include as much detail as you can — what you found, how to reproduce it, and what the potential impact might be.

I'll do my best to respond within a few days. If the vulnerability is confirmed I'll work on a fix and credit you in the release notes if you'd like. If it's not something I can reproduce or it falls outside the scope of the project I'll let you know.

Thanks for taking the time to report it.
